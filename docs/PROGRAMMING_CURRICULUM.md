# Computer Logic and Programming Curriculum (K-8th)

## Overview
Computer logic and programming concepts are integrated throughout the K-8 curriculum, starting with foundational sequential thinking in kindergarten and progressing to advanced algorithmic concepts in 8th grade.

## Grade-Level Scope

### Kindergarten-1st Grade
**Focus**: Sequential thinking, patterns, step-by-step instructions

**Lessons**:
1. **Step by Step** - Instructions happen in order, one step at a time
2. **Patterns Everywhere** - Computers find and follow patterns
3. **Find the Error** - Sometimes things go wrong and we need to find the mistake
4. **Follow the Directions** - Clear instructions help us complete tasks

**Key Concepts**: Sequence, pattern, error, instruction, step, first, then, next

### 2nd-3rd Grade
**Focus**: Basic logic (if/then), loops, variables, algorithmic thinking

**Lessons**:
1. **If This Then That** - Basic conditional logic
2. **Repeat the Pattern** - Loops repeat actions over and over
3. **Put It Together** - Complex tasks break into simpler steps
4. **True or False** - Statements can be correct or incorrect

**Key Concepts**: Condition, if, then, loop, repeat, true, false, decompose, step

### 4th-5th Grade
**Focus**: Algorithms, debugging, functions, arrays/lists, Boolean logic

**Lessons**:
- **Algorithms: Step-by-Step Solutions** - Clear sets of steps to solve problems
- **Debugging: Finding and Fixing Errors** - Process of finding and fixing mistakes
- **Functions: Reusable Blocks** - Named blocks of code that perform tasks
- **Arrays and Lists** - Store multiple values in order
- **Boolean Logic: AND OR NOT** - Combine conditions using Boolean operators
- **Variables and Data Types** - Different types need different storage
- **Loops: For and While** - Repeat code until condition is met
- **Pseudo-code: Planning Before Coding** - Plain language description of logic

**Key Concepts**: Algorithm, debug, function, array, list, boolean, and, or, not, variable, integer, float, string, iterate, condition

### 6th-8th Grade
**Focus**: Objects, classes, events, data structures, complexity, OOP

**Lessons**:
- **Objects and Properties** - Attributes and behaviors
- **Classes and Instances** - Blueprints versus specific objects
- **Events and Event Handlers** - Programs respond to events
- **Data Structures: Stacks and Queues** - Organize data in specific ways
- **Algorithm Complexity: Big O** - How algorithm performance scales
- **Recursion: Functions Calling Themselves** - Break problems into smaller versions
- **Searching and Sorting Algorithms** - Common algorithms for organizing data
- **Object-Oriented Design** - Encapsulation, inheritance, polymorphism
- **APIs and Modular Design** - Components communicate without exposing internals
- **Boolean Algebra and Logic Gates** - Logic gates implement Boolean operations
- **Problem Decomposition and Abstraction** - Breaking complex problems into pieces

**Key Concepts**: Object, class, instance, method, property, event, handler, stack, queue, recursion, Big O, logarithmic, quadratic, inheritance, polymorphism, API, encapsulation, abstraction, interface

## Integration with Grammar Spiral

Computer logic concepts are trained alongside grammar at each grade level:
- 4th grade: Algorithm, step, sequence, pattern, error, debug, function, loop, variable, array
- 5th grade: Boolean, and, or, not, true, false, condition, loop, iterate, variable
- 6th grade: Object, class, instance, method, property, event, handler, stack, queue, data
- 7th grade: Algorithm complexity, recursion, searching, sorting, debugging
- 8th grade: OOP, APIs, Boolean algebra, logic gates, decomposition, abstraction

## Assessment

Each grade level assessor includes `test_programming_logic_Xth()` function that tests:
- Node existence for key programming terms
- Connection counts to related concepts
- Integration with curriculum knowledge

## Training

Programming concepts are trained using `train_fast()` with 12 repetitions per grade level, integrated into the full pipeline alongside grammar and core curriculum corpora.
